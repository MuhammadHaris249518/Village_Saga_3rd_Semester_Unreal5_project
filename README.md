**Village Saga** is a 3D adventure game developed using **Unreal Engine 5** as a 3rd semester academic project.  
The player explores a mysterious village, collecting coins while avoiding **hidden traps** and surviving attacks from **enemy AI**. The main objective is to **collect 50 coins to win the game**.  

The game demonstrates skills in **level design, AI programming, and Unreal Engine 5 C++ and Blueprint integration**.

- **Player Exploration:** Navigate through a detailed village environment.  
- **Coin Collection:** Collect a total of 50 coins to complete the level.  
- **Enemy AI:** Enemies patrol the village and chase the player to reduce health.  
- **Hidden Traps:** Traps are scattered across the village to challenge the player.  
- **Multiple Interactions:** Players can interact with objects in the environment.  
- **Immersive 3D Environment:** Terrain, buildings, and assets are built using Unreal Engine 5 tools and assets.
